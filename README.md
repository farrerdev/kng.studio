# kng.studio

Mock customer catalog for KNG.studio homewear products.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy the repository to Vercel as a Vite project. The production build command is `npm run build` and the output directory is `dist`.

## Update shop links

Edit `src/config/shop.ts` and replace the placeholder Instagram/Facebook URLs with the real messaging links.
